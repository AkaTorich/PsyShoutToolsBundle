# Thunder Price Calculator

## Описание
Thunder Price Calculator - это профессиональный калькулятор для расчета параметров торговых позиций в криптовалютном трейдинге. Калькулятор автоматически рассчитывает оптимальный объем позиции на основе вашего риска, а также определяет уровни стоп-лосса и тейк-профита.

## Основные возможности
- Автоматический расчет объема позиции на основе риска
- Поддержка длинных (Buy) и коротких (Sell) позиций
- Автоматическое определение цены тика для разных криптовалют
- Расчет необходимой маржи и эффективного плеча
- Предупреждения о высоких рисках
- Адаптивные минимальные объемы для разных ценовых диапазонов

## Как использовать

### Входные параметры:
1. **Размер депозита ($)** - общий размер вашего торгового счета в USD
2. **Риск** - сумма в USD или процент от депозита, который вы готовы потерять в сделке
3. **Тип риска** - выберите USD для абсолютного значения или Percent для процента от депозита
4. **Текущая цена** - актуальная рыночная цена криптовалюты
5. **Цена тика ($)** - стоимость минимального изменения цены (определяется автоматически)
6. **Тип маржи** - Изолированная или Кросс-маржа
7. **Тип ордера** - Buy (длинная позиция) или Sell (короткая позиция)

### Выходные параметры:
- **Тейк-профит** - цена для фиксации прибыли
- **Стоп-лосс** - цена для ограничения убытков
- **Объем позиции** - количество монет/контрактов для покупки/продажи
- **Сумма сделки** - полная стоимость позиции в USD
- **Необходимая маржа** - сумма, которая будет заблокирована под позицию
- **Рекомендуемое плечо** - оптимальное кредитное плечо

## Автоматическое определение цены тика
Калькулятор автоматически определяет цену тика на основе текущей цены инструмента:
- Для BTC (> $10,000): $1 за пункт
- Для ETH (> $1,000): $0.1 за пункт
- Для монет > $100: $0.01 за пункт
- Для монет > $10: $0.001 за пункт
- Для монет > $1: $0.0001 за пункт
- Для монет < $1: $0.00001 за пункт

## Формулы расчета

### Объем позиции:
```
Volume = RiskAmount / (StopDistance × TickPrice)
```

### Стоп-лосс:
- Для Buy: CurrentPrice × (1 - 0.01)
- Для Sell: CurrentPrice × (1 + 0.01)

### Тейк-профит:
- Risk:Reward = 1:2 (по умолчанию)
- Для Buy: CurrentPrice + (StopDistance × 2)
- Для Sell: CurrentPrice - (StopDistance × 2)

## Предупреждения
Калькулятор автоматически предупреждает о:
- Корректировке объема до минимально допустимого
- Высоком риске (> 5% от депозита)
- Высоких требованиях к марже (> 50% от депозита)

## Системные требования

- .NET Framework 4.7.2 или выше
