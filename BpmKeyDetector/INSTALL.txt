🎉 НИКАКОЙ УСТАНОВКИ НЕ ТРЕБУЕТСЯ!

Это приложение полностью автономное (self-contained) и уже
содержит все необходимые компоненты .NET Runtime внутри себя.


📦 ЧТО НУЖНО СДЕЛАТЬ:

1. Откройте папку "publish"
2. Найдите файл "BpmKeyDetector.exe"
3. Запустите его двойным кликом
4. Наслаждайтесь!


✨ СИСТЕМНЫЕ ТРЕБОВАНИЯ:

✓ Windows 10 или Windows 11 (64-bit)
✓ Минимум 4 GB RAM (рекомендуется 8 GB)
✓ 100 MB свободного места на диске
✓ .NET Runtime НЕ ТРЕБУЕТСЯ!


🎵 ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:

✓ MP3
✓ WAV
✓ AIFF/AIF
✓ M4A
✓ FLAC


🚀 КАК ПОЛЬЗОВАТЬСЯ:

1. Нажмите "ВЫБРАТЬ ПАПКУ" и укажите папку с музыкой
2. Нажмите "АНАЛИЗИРОВАТЬ" для начала обработки
3. Дождитесь окончания анализа (прогресс показывается)
4. Выберите действие:
   • "ПЕРЕИМ. ТАМЖЕ" - переименовать в той же папке
   • "ПЕРЕИМ. В ПАПКУ" - скопировать с новыми именами


📝 ФОРМАТ РЕЗУЛЬТАТА:

До:   Track Name.mp3
После: Track Name [BPM 128.0 - Key A min].mp3


💡 СОВЕТЫ:

• Для лучших результатов используйте файлы хорошего качества
• Анализ может занять время для больших библиотек
• Приложение анализирует середину трека для точности
• Можно закрыть и перезапустить в любой момент

📞 ПОДДЕРЖКА:
