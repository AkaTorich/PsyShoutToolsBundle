# TaskReminderApp - Планировщик задач с конвертером раскладки

WPF приложение на C# 8 для планирования задач с напоминаниями и автоматической конвертацией текста между русской и английской раскладками клавиатуры.

## Современный дизайн

Приложение имеет современный пользовательский интерфейс в темных тонах:

- **Темная тема:** Приятная для глаз цветовая схема в темных тонах
- **Современные элементы:** Скругленные углы, тени, градиенты
- **Визуальная обратная связь:** Анимированные индикаторы статуса и интерактивные элементы
- **Интуитивный интерфейс:** Четкое разделение секций и яркие акцентные цвета
- **Красивый заголовок:** Градиентный хедер с информацией о дате и времени
- **Стильные карточки задач:** Задачи отображаются в виде красивых карточек с тенями

## Возможности

### 1. Планировщик задач
- Добавление задач с названием, описанием и временем напоминания
- Автоматические уведомления в указанное время
- Отметка задач как выполненных
- Удаление задач
- Сохранение задач между сеансами

### 2. Конвертер раскладки клавиатуры
- Глобальный хук клавиши F2
- Автоматическое определение раскладки текста
- Конвертация выделенного текста между русской и английской раскладкой
- Поддержка всех символов, букв и цифр
- Управление хуком (запуск/остановка)

## Как использовать

### Запуск приложения

1. Откройте проект в Visual Studio 2022 или новее
2. Убедитесь, что установлен .NET 8.0 SDK
3. Запустите проект (F5)

Или через командную строку:
```bash
dotnet build
dotnet run
```

### Работа с задачами

1. **Добавление задачи:**
   - Введите название задачи
   - Добавьте описание (опционально)
   - Выберите дату напоминания
   - Укажите время (час и минуты)
   - Нажмите "Добавить задачу"

2. **Управление задачами:**
   - Выберите задачу в списке
   - "Отметить выполненной" - пометить как завершенную
   - "Удалить выбранную" - удалить задачу
   - "Обновить список" - обновить отображение

3. **Напоминания:**
   - Приложение автоматически показывает уведомления
   - Проверка напоминаний каждые 10 секунд
   - Уведомления появляются только один раз для каждой задачи

### Конвертер раскладки

1. **Запуск хука:**
   - Нажмите кнопку "Запустить хук"
   - Статус изменится на "Активен" (зеленый)

2. **Использование:**
   - Выделите текст в любом приложении
   - Нажмите клавишу F2
   - Текст автоматически конвертируется и заменяется

3. **Примеры конвертации:**
   - `ghbdtn` → `привет` (английская неправильная → русская правильная)
   - `руддщ` → `hello` (русская неправильная → английская правильная)
   - Работает с большими буквами, цифрами и спецсимволами

4. **Остановка хука:**
   - Нажмите кнопку "Остановить хук"
   - Хук перестанет отслеживать нажатия F2

## Технические детали

- **Платформа:** .NET 8.0 Windows
- **UI Framework:** WPF
- **Язык:** C# 8.0
- **Дизайн:** Современный темный интерфейс с градиентами и эффектами
- **Сохранение данных:** JSON файлы в `%AppData%\TaskReminderApp\tasks.json`
- **Хук клавиатуры:** Windows API (user32.dll)

## Цветовая палитра

Приложение использует современную цветовую схему:

- **Фон:** Градации серого (#1E1E1E, #252526, #2D2D30)
- **Акценты:**
  - Cyan (#06B6D4) - основные элементы
  - Blue (#0E7490) - кнопки действий
  - Green (#10B981) - успешные операции
  - Red (#EF4444) - удаление и ошибки
  - Orange (#F97316) - предупреждения
- **Текст:** Белый (#FFFFFF) и серый (#CCCCCC)
- **Границы:** Темно-серый (#3F3F46)

## Возможные проблемы

1. **Хук не работает в некоторых приложениях:**
   - Некоторые приложения с повышенными правами могут блокировать хуки
   - Попробуйте запустить приложение от имени администратора

2. **Конвертация не работает:**
   - Убедитесь, что хук запущен (статус "Активен")
   - Выделите текст перед нажатием F2
   - Проверьте, что текст скопировался в буфер обмена

3. **Напоминания не приходят:**
   - Убедитесь, что приложение запущено
   - Проверьте правильность даты и времени задачи
   - Сервис напоминаний работает в фоновом режиме
