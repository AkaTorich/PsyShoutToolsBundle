# PsyShoutTools Bundle

Коллекция из 35+ профессиональных Windows-утилит для работы с аудио, задачами и медиа-контентом.

## Основные возможности

### Аудио инструменты
- **BpmKeyDetector** - определение BPM и музыкального ключа с Material Design UI
- **AudioConverter** - конвертация аудио форматов
- **PsyTrancePlayer** - музыкальный плеер для Psytrance
- **WavTagEditor** - редактор метаданных WAV
- **ScaleHelperVST** - VST плагин для работы с музыкальными гаммами

### Генераторы
- **ARPGenerator** - генератор арпеджио
- **PADGenerator** - генератор PAD синтеза
- **BrainWaveGenerator** - генератор бинауральных ритмов

### Управление задачами
- **TasksReminder** / **TaskCalendar** / **TaskManWPF** - несколько вариантов менеджеров задач
- **PomedoroWorker** - таймер по технике Pomodoro

### Утилиты
- **ImageResizer** - изменение размера изображений
- **MultiConverter** - мультиформатный конвертер
- **NetworkMonitor** - мониторинг сети
- **BackupManager** - управление резервными копиями
- **Archiver** - архиватор файлов

## Технологии

- **C#** (.NET Framework, .NET 8/9, WPF, Windows Forms)
- **C++** (производительно-критичные компоненты)
- **NAudio** (v2.2.1) - обработка аудио
- **MaterialSkin.2** - Material Design UI
- **BASS Audio Library** - продвинутый аудио движок

## Сборка

Требуется **Visual Studio 2022** (v17.11+) с поддержкой C# и C++.

```bash
# Открыть решение
PsyShoutToolsBundle.sln

# Собрать все проекты
Build > Build Solution
```

Скомпилированные приложения будут в папке `bin/`.
